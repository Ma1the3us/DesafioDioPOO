# Desafio de Abstração com POO

Este projeto foi desenvolvido como parte de um desafio de bootcamp, com o objetivo de aplicar os conceitos de **Programação Orientada a Objetos (POO)**.

## 🚀 Objetivo
Praticar a abstração de um domínio utilizando os quatro pilares da POO:

- **Abstração**
- **Encapsulamento**
- **Herança**
- **Polimorfismo**

## 🧠 Conceitos Aplicados
O projeto simula um cenário onde entidades do mundo real são representadas por classes, permitindo uma melhor organização e reutilização de código.

## 🛠️ Tecnologias Utilizadas
- Java
- IntelliJ IDEA

## 📚 Aprendizados
Durante o desenvolvimento, foi possível compreender melhor como estruturar sistemas orientados a objetos e como os pilares da POO se complementam na construção de aplicações mais robustas.
